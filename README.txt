
# Portfolio Phase 3 (GitHub Fetcher Added)

This is a personal portfolio website built using **HTML, CSS, and JavaScript**. In this version, a **GitHub Profile Fetcher** feature has been added to display GitHub user information dynamically.

## Features
- Responsive portfolio layout.
- **GitHub Profile Fetcher:** Enter a GitHub username to fetch and display profile information.
- Styled GitHub card with user avatar, name, bio, and other details.

## Files Updated
- **index.html:** Added a section for GitHub username input.
- **css/styles.css:** Added styling for the GitHub profile card.
- **js/script.js:** Fetches profile data from GitHub API and displays it dynamically.

## How to Use
1. Open `index.html` in your browser.
2. Scroll to the **"GitHub Profile Fetcher"** section.
3. Enter a GitHub username (e.g., `octocat`) and click **Fetch**.
4. The profile information will be displayed below the input.

## Technologies Used
- HTML
- CSS
- JavaScript (Fetch API)
